# 🐾 [기획서] 포차코의 오늘 할 일 캘린더 웹앱 (Pochacco Todo Calendar)

본 기획서는 사용자 친화적이고 직관적인 UI/UX를 바탕으로, 특정 날짜별 할 일(Todo)을 관리할 수 있는 파스텔톤 포차코 테마의 단일 페이지 웹 애플리케이션(SPA) 구축을 목적으로 합니다.

---

## 1. 프로젝트 개요

### 1.1 개발 목적 및 배경
* **동기 부여 중심의 도구**: 딱딱한 텍스트 위주의 기존 할 일 관리 앱에서 벗어나, 귀여운 캐릭터와 상호작용 피드백을 결합하여 하루 계획 수립에 재미를 부여합니다.
* **학습 및 사용 용이성**: 초등학생 및 프로그래밍 초보자도 쉽게 구조를 파악하고 커스텀할 수 있도록 외부 파일 의존성을 최소화하고, 단일 HTML 파일(HTML/CSS/JS) 구조로 설계합니다.

### 1.2 타겟 유저
* 귀여운 캐릭터(산리오 포차코 등)를 좋아하는 사용자
* 하루의 할 일을 직관적으로 기록하고 시각적인 달성을 확인하고 싶은 어린이 및 학생
* 무거운 설치형 프로그램 대신 브라우저에서 가볍게 쓸 수 있는 메모장이 필요한 사용자

---

## 2. 핵심 기능 요구사항 (Functional Requirements)

### 2.1 포차코 마스코트 & 상호작용 (Interaction)
* **SVG 기반 가볍고 선명한 그래픽**: 별도 이미지 파일(.png, .jpg) 없이 코드만으로 포차코를 구현하여 로딩 속도를 최적화하고 해상도 저하를 방지합니다.
* **실시간 대화 피드백**: 사용자의 행동(추가, 완료, 삭제, 날짜 변경)에 따라 상단 말풍선의 텍스트가 실시간으로 변경됩니다.
* **이스터 에그(Wink)**: 캐릭터 클릭 시 오른쪽 눈 모양 변형(ry, rx 속성 변경)을 통해 윙크 애니메이션과 전용 대사를 출력합니다.

### 2.2 접이식 캘린더 (Calendar)
* **연/월 탐색 기능**: 이전 달(<) 및 다음 달(>) 이동 버튼을 통해 자유롭게 날짜를 탐색합니다.
* **선택 날짜 하이라이트**: 사용자가 현재 바라보고 있는 날짜를 민트색 원과 입체 그림자로 강조 표시합니다.
* **날짜별 상태 표시등 (Dot Indicator)**:
  * 미완료 할 일이 있는 날짜: 분홍색 점(`•`) 표시
  * 해당 날짜의 모든 할 일을 완료한 날짜: 초록색 체크 점(`•`) 표시
* **토글(Toggle) 인터페이스**: 모바일 화면 공간 확보를 위해 달력 영역을 펼치고 접을 수 있는 아코디언 버튼을 제공합니다.

### 2.3 할 일 관리 (Todo 리스트)
* **날짜별 독립된 데이터 바인딩**: 선택된 날짜(`YYYY-MM-DD`)에 종속된 할 일 목록만 필터링하여 동적으로 렌더링합니다.
* **글자 수 제한**: UI 깨짐 방지를 위해 입력값은 최대 30자로 제한합니다.
* **예외 처리**: 공백 입력 후 추가 시 브라우저 내장 alert 대신 하단 커스텀 토스트(Toast) 팝업으로 경고를 노출합니다.
* **커스텀 체크박스**: 기본 체크박스를 숨기고 애니메이션이 가미된 원형 체크 UI를 적용합니다. 완료 시 텍스트에 취소선(분홍색 포인트 라인)을 긋고 투명도를 조절합니다.

### 2.4 게이미피케이션 & 성취 보상
* **파티클 폭죽 효과**: 할 일 체크 시 화면 하단 중앙에서 파스텔톤 가루가 사방으로 튀는 Canvas 애니메이션을 구현합니다.
* **올 클리어(All Done) 보상**: 특정 날짜의 모든 일정을 완료하면 3연속 폭죽 연출과 함께 포차코의 극찬 멘트가 등장합니다.

### 2.5 데이터 지속성 (Persistence)
* **로컬 스토리지(LocalStorage)**: 새로고침하거나 브라우저를 종료해도 데이터가 유지되도록 JSON 데이터 포맷으로 브라우저에 자동 저장합니다.
* **하위 호환성(Migration)**: 구버전 데이터(날짜 정보가 없는 옛날 투두 리스트) 발견 시, 유저 데이터를 잃지 않고 자동으로 오늘 날짜 일정으로 안전하게 마이그레이션합니다.

## 3. UI/UX 디자인 가이드 (Design System)

### 3.1 컬러 팔레트 (Color Palette)
| 용도 | 색상 코드 (Hex) | 디자인 느낌 |
| :--- | :--- | :--- |
| 메인 배경색 | `#F3FBF7` | 부드럽고 차분한 파스텔톤 민트 그린 |
| 메인 카드 배경 | `#FFFEEF` | 따뜻하고 안락한 느낌의 크림 옐로우 |
| 포인트 컬러 1 | `#00BCD4` | 시원하고 직관적인 사이안 민트 (선택 날짜, 카운터) |
| 포인트 컬러 2 | `#FF8A8A` | 포차코의 옷색이자 중요 알림용 파스텔 레드 (삭제, 체크 점) |
| 포인트 컬러 3 | `#FFE082` | 귀여운 스티커 느낌의 파스텔 옐로우 (추가 버튼) |
| 테두리 및 텍스트 | `#333333` / `#37474F` | 만화책 느낌의 진하고 부드러운 차콜 블랙 |

### 3.2 타이포그래피 (Typography)
* **메인 서체**: `Gowun Dodum` (가독성이 높고 단정한 느낌의 고딕체)
* **포인트 서체**: `Jua` (둥글고 오동통하여 타이틀, 카운터, 버튼 등에 최적화된 서체)

### 3.3 레이아웃 특징
* **카툰 보더 스타일 (Cartoon Border)**: 모든 주요 요소에 `4px` 또는 `2px` 실선 테두리를 부여하고, Y축으로만 치우친 진한 그림자(`box-shadow: 0 8px 0px #333`)를 적용하여 스티커북 같은 2.5D 만화 효과를 연출합니다.
* **반응형 웹 디자인**: 테일윈드 CSS(Tailwind CSS)의 가변 그리드와 패딩 시스템을 사용하여 데스크톱, 태블릿, 모바일 화면에 모두 유연하게 대응합니다.
